Fusion Behavioral Care

Wix • 2024

Overview

A patient-first website for a mental health medical practice, built on Wix. The goals were to make key actions obvious (Call, Directions, Patient Portal) and improve readability, mobile performance, and basic accessibility.

My Role

  • Built the site with Wix Editor using a theme as a base; customized layout and navigation
  • Created and edited content; coordinated with the client to ensure accuracy
  • Ongoing site management and updates
  • Applied accessibility basics: alt text, color contrast, semantic landmarks
  • Configured SEO essentials: titles, meta descriptions, and open graph

Highlights

  • Clear “top three” actions: Call • Directions • Patient Portal
  • Calmer visual hierarchy and readable content blocks for patients
  • Representative Lighthouse checks showed gains in performance and accessibility
