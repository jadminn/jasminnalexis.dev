Mother's Meadow

WordPress • June 2024

Overview

A WordPress website created to launch and promote a new pick-your-own blueberry farm in Brandon, Vermont — providing visitors with information, location guidance, and seasonal updates.

My Role

  • Set up the domain and configured hosting
  • Built the full website using WordPress themes, editor, and custom layout decisions
  • Created all written and visual content using Canva and Photoshop
  • Shot original photography on-site
  • Worked directly on the farm — personally helped plant over 4,000 blueberry bushes

Highlights

  • Interactive picking map and educational blog resources for visitors
  • Mobile-friendly layout for local visitors accessing the site from the fields
  • Clean, easy-to-navigate experience for first-time guests
