Pong — JavaScript Clone

HTML • CSS • JavaScript

Overview

A browser-based Pong remake with colorful visuals and optional background music. Supports 1-player (AI opponent) and 2-player modes, written in vanilla JavaScript with a clean, modular structure.

How to Play

  • 1P: W/S to move
  • 2P: W/S (left) · ↑/↓ (right)
  • Press Play to start (unlocks audio per browser autoplay rules)

Highlights

  • Modular game loop + object model (paddles, ball, collision, scoring)
  • Deterministic physics with speed ramp & serve logic
  • Input handling, pause/reset states, and simple AI for 1P mode
  • Audio gated behind user gesture for reliable playback on the web

Live Code

Tech Notes

  • Canvas-based rendering; fixed timestep game loop
  • Keyboard events with graceful focus (click canvas once if needed)
  • Audio paths use ./sfx/... and lowercase filenames for GitHub Pages
