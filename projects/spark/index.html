Spark Community Solutions

Wix • 2025

Overview

Spark Community Solutions is a medical practice management company that wanted a fresh, cohesive web presence without rebuilding from scratch. My goal was to enhance visual identity, improve readability across devices, and unify brand elements such as typography, spacing, and palette while working fully inside the Wix Editor and honoring previously used content.

My Role

  • Refined visual design system (fonts, hierarchy, spacing)
  • Adjusted layout for clarity and mobile-first usability
  • Re-edited images to align with the new green-focused branding
  • Implemented gentle button animations, graphics and hover states

Highlights

  • Unified typography and green tonal palette to improve brand consistency
  • Refreshed existing imagery to better convey professionalism
  • Created reusable styled buttons
  • Enhanced user flow for new visitors looking for information

Next Steps

Upcoming improvements include refining the testimonial system (native vs. embedded reviews).
